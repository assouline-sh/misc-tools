# misc-tools

- **bionic-pdf** — Converts a PDF into a bionic-reading PDF by bolding the first half of each word while preserving the original layout.
- **ctfdump** — Scrapes all challenges and files from a CTFd or rCTF instance into organized local directories.
- **spotiback** — Backs up and restores Spotify playlists, liked songs, and followed artists as JSON snapshots.
